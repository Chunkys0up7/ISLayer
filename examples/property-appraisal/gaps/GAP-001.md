# GAP-001: Missing Automated Comparable Sales Validation Against MLS Data

## Description

The Assess Property Value task (PA-ASV-001) currently relies on the comparable sales data as reported by the appraiser in the UAD XML without independent verification against MLS (Multiple Listing Service) or public records data. This means the lender cannot automatically confirm that the reported sale prices, sale dates, and property characteristics for comparable sales are accurate.

## Impact

- **Risk**: Inflated or fabricated comparable sales could support an artificially high appraised value, increasing collateral risk.
- **Compliance**: Fannie Mae Selling Guide B4-1.3 recommends that lenders independently verify comparable sales data when feasible.
- **Operational**: Without automated cross-referencing, manual verification by the appraisal desk is required for high-risk loans, adding 1-2 business days to the process.

## Current Workaround

For loans flagged with adjustment threshold violations or value inflation indicators, the appraisal desk manually checks comparable sales against MLS data. This is a labor-intensive process that is not applied consistently to all appraisals.

## Proposed Resolution

1. Integrate a MLS data provider API (e.g., CoreLogic, Black Knight, ATTOM) into the value assessment pipeline.
2. For each comparable sale reported in the UAD XML, query the MLS API to verify the sale price, sale date, and property characteristics.
3. Flag discrepancies exceeding defined tolerances (e.g., sale price variance greater than 5%, sale date mismatch greater than 30 days).
4. Add a "comparables_verified" flag to the value assessment output schema (ICT-PA-ASV-001).
5. Update the process to route unverified or discrepant comparables to the manual review path.

## Dependencies

- MLS data provider contract and API credentials (vendor evaluation in progress).
- Updates to ICT-PA-ASV-001 contract schema (non-breaking: adding optional output field).
- Updates to CAP-PA-ASV-001 procedure and business rules.

## Notes

- Some MLS data providers have licensing restrictions on automated querying; legal review is required.
- Public records data (county assessor) may supplement MLS data for sale price verification but lacks detailed property characteristics.
